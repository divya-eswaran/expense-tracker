Modern Expense Tracker:

A sleek, responsive, and feature-rich Expense Tracker web application. This tool helps users manage their finances by logging daily expenses, categorizing spending, and visualizing data through interactive charts.

✨ Features

📊 Dynamic Data Visualization: Real-time pie chart integration using Chart.js to see spending distribution at a glance.

🌓 Smart Dark Mode: Toggle between light and dark themes with persistent storage (remembers your preference).

💾 Local Storage Persistence: Your data is saved locally in your browser—no login required, and your data stays private.

🔍 Live Search & Filter: Instantly find specific transactions by description.

✏️ Full CRUD Operations: Easily Add, Read, Update, and Delete expenses.

📱 Fully Responsive: Optimized for mobile, tablet, and desktop viewing.
